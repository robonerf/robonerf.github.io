Overview

Workshop Details


Welcome to the CORL 2023 Workshop on Neural Fields in Robotics!
The ”Neural Fields in Robotics” workshop aims to explore the role and potential of implicit neural representations (INRs) and NeRFs in various robotics domains, including 6D object pose estimation, simultaneous localization and mapping (SLAM), manipulation with reinforcement learning (RL), object reconstruction, physics modeling, and planning/navigation. By leveraging recent advancements in computer vision, such as neural radiance fields (NeRFs) and deep Signed Distance Functions (DeepSDFs), this workshop aims to foster discussions and collaborations in the robotics community regarding the impacts and possibilities of INRs.
Through invited talks and a poster session, the workshop will address several key topics:
  • The advantages of implicit neural scene representations beyond reconstruction for the robotics context
  • Trade-offs between an implicit and explicit representation for robotics applications like manipulation/SLAM and navigation
  • Online training of neural fields for robotics and the speed/efficiency trade-offs
  • Balancing fully-implicit representations which are less interpretable with hybrid neural field-based representations which offer more interpretability such as voxel grid-based reconstruction

This workshop would encourage these main questions and foster much more discussion on the breadth (application) and depth (algorithms development) of neural fields for leveraging INRs and Neural Fields for tasks like localization, motion planning, manipulation, locomotion, semantic scene understanding as well as adapting INRs to simulate and predict dynamic and uncertain scenes.

Paper submission

Call for papers

📚 Topics



The RoboNeRF workshop accepts papers in relation with application of neural fields and implicit representation to robotics areas. The workshop will include a poster and an oral session for related works. Topics of this workshop include but are not limited to: 1.

  • Neural Fields for Real-time Robotics Appplications
  • NeRFs and Implicit Representations for SLAM
  • Implicit Grasping for manipulation
  • 6D pose estimation using implicit fields
  • Mobile robot navigation using implicit functions
  • Efficient Hybrid Representations for 3D Scene Understanding
  • Articulated object reconstruction and manipulation
  • Nueral Implicit Data Generation and capture of 3D scenes
  • Tactile and Deformable object manipulation using learnt implicit models
  • Physical an dynamical propoerties estimation based neural field models
  • ...
The submitted 4-page abstracts will be peer-reviewed in CORL format. Abstracts will be presented in the workshop poster session, and a portion of the accepted papers will be orally presented.
